Me

夜阑卧听风吹雨，铁马冰河入梦来。

我是一名来自湖南的初中生，一个编程和FPS、竞速游戏的爱好者。

目前在一所半军事化学校就读，管的倒不严，问题是一个月放一次月假，而且还就只有4天左右，这就有点难受了，而且极其无聊，每天就跟循环一样。

希望初三中考能考到自己心仪的高中。 😎

### 下面是我的一些小经历🐹

2018 🍔

这时候的我刚对编程萌发兴趣，那时候我还没有电脑，只有一部手机，那时候我特别想开发一款属于我自己的APP或者游戏。我当时经常在B站上看到一些30分钟速建游戏或者一个学生耗费XX时间做的游戏的那种类型的视频（大多数好像都是Unity），这使我萌发了自己做游戏的心愿，于是我便开始看一些游戏引擎之类的教学视频，大都是Unity、虚幻4，好像还有RPG Maker，但当时我什么也看不懂，也没有去学习的条件，毕竟只有一台手机，但我依然觉得这些东西很酷，很有意思。当时我特别爱玩FPS游戏，我当时的梦想就是做一款开放世界类的，带有少量跑酷元素及技能元素，有战役和线上可玩，未来风格的一款FPS游戏，可惜在当时看来简直就是不可能。

2019 🌭

这时候终于有了一台配置不错的电脑，这也是我梦的起源，终于可以玩上一直想玩的CSGO了（现在我大概有1000小时+了），这时我也开始陆陆续续的接触了编程，我在那时候便开始学习了Python，现在想想，Python的基础实在很简单，没必要花太多时间去教学，可惜当时我看到的教学都是100或80小时+的，但我还是学了下去（当然没有学完），这时候的我终于知道了编程的乐趣，以及享受它所带来的成就感，这是独一无二的。

2020 🍕

在2020，我开始把我的目光投向GUI开发，我用Winform做出了属于我的第一个GUI程序，我的UI审美也在不断提高，后面我又学了C#和WPF，不得不说.NET做GUI真是一种很惬意的体验，WPF是我认为所有GUI框架中做出来界面最漂亮的，而且也很简单，听说.NET 6和Visual Studio 2022正式版也快来了，不过.NET 6的程序还是不能在Linux上运行，这让我感到有些可惜。后面我可能还会去学下Qt什么的（为什么是C++啊啊啊）

现在 🎉

现在的我已经学了C语言基础、Python、HTML + CSS + JS + Vue.js、C#这几项，对编程和操作系统也有了自己一定的见解，后续我准备学一下Rust。
